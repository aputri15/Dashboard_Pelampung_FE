Upload Data Transaksi

Upload file Excel data transaksi tahunan ke sistem database.

Perbaiki data langsung di spreadsheet sumber. Jika file yang sama pernah di-upload, buka Riwayat Log Upload untuk mengizinkan upload ulang.

Panduan Upload

  • Pastikan file dalam format Excel (.xlsx atau .xls).
  • Sheet transaksi wajib bernama MASTER + 4 digit tahun, contoh MASTER2025.
  • File hanya boleh memiliki satu sheet transaksi dengan format nama tersebut.
  • Semua kolom transaksi wajib lengkap sebelum upload.
  • Total_Harga boleh kosong karena sistem menghitung otomatis dari Qty * Harga_Satuan.
