Upload Data Transaksi

Upload file Excel data transaksi bulanan ke sistem database.

Tarik dan lepas file Excel di sini

Atau klik untuk memilih file (format .xlsx, .xls) max 10MB

Panduan Upload

  • Pastikan file dalam format Excel (.xlsx atau .xls).
  • Data harus memiliki kolom yang wajib seperti: Nomor PO, Tanggal, Pelanggan, Produk, Qty, Harga.
  • Format tanggal bebas, sistem akan menyimpannya sebagai teks.
  • Sistem akan otomatis membersihkan dan memvalidasi format angka sebelum menyimpan ke database.
